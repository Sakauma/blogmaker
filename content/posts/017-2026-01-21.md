---<p><strong>Без меня</strong></p>
<p>На рассвете — без меня<br>На кассете — без меня<br>Без меня — за дверь, без меня — домой<br>Без меня — теперь, без меня —<br>анекдот с бородой навсегда</p>
<p>И убегает мой мир<br>Убегает земля<br>Бежит далеко-далеко<br>Куда-то далеко-далеко<br>И убегает мой мир<br>Убегает земля<br>Бежит далеко-далеко<br>Без оглядки далеко-далеко…</p>
<p>Корка хлеба без меня<br>Пальцем в небо — без меня<br>Без меня — апрель, без меня — январь<br>Без меня — капель, без меня —<br>отрывной календарь на стене</p>
<p>И убегает мой мир<br>Убегает земля<br>Бежит далеко-далеко<br>Куда-то далеко-далеко<br>И убегает мой мир<br>Убегает земля<br>Бежит далеко-далеко<br>Навек далеко-далеко…</p>
<p>Добрый ослик без меня<br>Горький дождик без меня<br>Без меня — сирень, без меня — герань<br>Без меня моя тень, без меня —<br>поздравления оттуда сюда</p>
<p>И убегает весь мир…<br>И убегает мой мир…</p>

<p><strong>没有我</strong></p>
<p>在黎明时分，没有我<br>在录音带上，没有我<br>门的后面，没有我<br>回到家里，没有我<br>就在现在，没有我<br>老旧笑话，没有我</p>
<p>我的世界在逃离<br>大地它也在逃离<br>它远远地逃离<br>直到天涯海角<br>我的世界在逃离<br>大地它也在逃离<br>它远远地逃离<br>头也不回离去</p>
<p>面包皮里，没有我<br>指向天空，没有我<br>四月里，没有我<br>一月里，没有我<br>水滴里，没有我<br>撕碎的挂历里，没有我</p>
<p>我的世界在逃离<br>大地它也在逃离<br>它远远地逃离<br>直到天涯海角<br>我的世界在逃离<br>大地它也在逃离<br>它远远地逃离</p>
<p>永远不再回来</p>
<p>好驴子上，没有我<br>苦涩的雨，没有我<br>丁香花里，没有我<br>天竺葵里，没有我<br>我的影子里，没有我<br>从此岸到彼岸的问候里，没有我</p>
<p>整个世界在逃离……<br>我的世界在逃离……</p>
